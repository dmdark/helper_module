=== Модуль добавления ключевых слов к хлебным крошкам ===

Данный модуль не зависит от модуля хлебных крошек и может быть использован как для крошек SEO модуля, так и для крошек, формируемых системой.

В config.php модуль включается:
'module_breadcrumbs_kw_enabled' => 1,

Для использования необходимо в шаблоне страницы выделить блок крошек тегами:
<!--$$crumbskw-->
	<!--Код блока хлебных крошек-->
<!--/$$crumbskw-->

Также необходимо указать код шаблонов в
/_seo/templates/breadcrumbs/
- container.html - общий контейнер, плейсхолдер {{items}} для указания дочерних элементов
- parents.html - шаблон для родительских ссылок
- item.html - шаблон для текущего элемента
Для parents.html и item.html доступны следующие плейсхолдеры:
- {{link}} - ссылка
- {{title}} - атрибут title
- {{text}} - текст крошки
